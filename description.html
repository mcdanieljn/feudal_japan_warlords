Feudal Japan Warlords


Created by Rolf Larsson

Inspired by Shogun, Samurai Swords, Ikusa boardgames, this map is done in tripleastyle, adding
Ships, Seazones, Cavalry and changes a few other things.

Features


  • 2, 3, 4 or 5 players, free for all
  • Rotating turnorder
  • Ninjas and Ronins
  • Daimyo experience
  • Guardvalues vs. assassination attempts
  • Boardgame like maximums for units
  • Random dealt territories or picked from start
  • and some more...
